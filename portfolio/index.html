About

Kyubin Han is a self-taught developer who have been taking various courses from MOOC platforms. He loves software developing because he can contribute to society by building applications that can help people.

He developed a vision assistance tool called P-eye for people who are visually impaired. He also built and launched a website where people can collaboratively translate documents for immigration to Canada.

Cloud Service for P-eye

The cloud service handles image processing including cropping images, finding contours, and image recognition using OpenCV and Google Cloud Vision API. A lightweight Python web framework, Flask, is used to build RESTful APIs and it is deployed on Amazon Web Services. The Raspberry Pi communicates with the cloud service through HTTP protocol POST in order to analyze the contents of images taken by users.


P-eye is a pair of camera-mounted glasses that utilize cloud computing services and machine learning technologies to analyze image content and relay it back in an audio format. It is designed for people who are visually impaired to recognize objects, printed words, and facial expressions.

Android Application for P-eye

P-eye with Bluetooth features -dubbed “BlueP-eye”- makes use of the Android mobile environment to enhance the capabilities of the P-eye. The custom designed user interface controls Bluetooth and wireless communication to interact with the P-eye and the cloud service deployed on AWS.

The Android app receives an encoded string of the image from P-eye through Bluetooth, sends the string to the web service through HTTP request to analyze image content, gets the result, and relay it back in an audio format using TTS.

Translate Together

Translate Together is a website where people can collaboratively translate documents provided from Canada Immigration's website and share with others.

The website is built on Firebase which offers real-time database and built-in authentication. AngularJS is chosen for front-end side in order to make use of MVVM design.

The website saves user's IP addressees in user history, maintains a blacklist to protect contents from disruption or vandalism, and provides additional features to the administer (deleting translations, banning users). Users can also report faulty translations
